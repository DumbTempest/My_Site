# Personal Website

My Personal website with Html, Css, using Alpinejs, Daisy ui and Tailwind.  


## Project Structure

```
Portfolio/
├── assets/          # Images and media
├── contact/         # Contact page
├── css/            # Stylesheets
├── js/             # JavaScript files
├── partials/       # Reusable components
├── projects/       # Projects page
├── resume/         # Resume page
└── index.html      # Main one
```


## Using This Template

1. Fork this repository
2. Rename it in Settings → Repository name
3. Update personal info in all pages.
4. Customize projects, resume, and contact pages
5. Deploy to GitHub Pages, Netlify, or Vercel
